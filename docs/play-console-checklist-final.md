# Checklist Final - Play Console (Copiar e Colar)

Atualizado em: 07/05/2026
Aplicativo: Demanda Controller

Este checklist foi criado para concluir rapidamente o preenchimento no Play Console usando os dados ja validados do app.

## 1. URL de politica de privacidade

Cole exatamente esta URL no campo obrigatorio:

- https://vander27.github.io/demanda_controller/privacy-policy.html

URL complementar de termos:

- https://vander27.github.io/demanda_controller/terms-license.html

## 2. Publico-alvo

Marcar como:

- Nao direcionado a criancas
- Publico profissional/adulto
- Sem apelo infantil

Texto curto para justificativa:

"Demanda Controller e um aplicativo de uso profissional para gestao operacional e financeira. Nao foi desenvolvido nem promovido para criancas."

## 3. Classificacao de conteudo

Respostas recomendadas:

- Violencia: nao
- Conteudo sexual/nudez: nao
- Linguagem ofensiva: nao
- Drogas/alcool/tabaco: nao
- Apostas/jogos de azar: nao

## 4. Data Safety (resumo objetivo)

Marcar:

- Coleta dados: sim
- Compartilha dados: sim (apenas provedores essenciais)
- Criptografia em transito: sim
- Exclusao de dados pelo usuario: sim
- Publicidade: nao
- Venda de dados: nao
- Rastreamento entre apps/sites: nao

Dados a declarar:

- E-mail (autenticacao)
- Dados operacionais inseridos pelo usuario (empresas, sites, adiantamentos, lancamentos, relatorios, configuracoes)

Finalidades:

- Funcionalidade do app
- Gerenciamento de conta
- Backup/restauracao/sincronizacao
